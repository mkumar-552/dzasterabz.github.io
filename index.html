About PyMRCE

PyMRCE is a Non-Profit & Open Learning Community. Under the Python Express Initative by the Python Software Society of India (PSSI), & the supervision of Hyderabad Python User Group (HydPy), PyMRCE serves as the Python User Community for Under-Graduation students in order to spread the uses and potential of this Modern Language. It also provides the students/members with a wide professional network & support, helping them to achieve their dreams in different fields of IT.
